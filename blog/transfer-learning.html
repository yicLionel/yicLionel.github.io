Transfer Learning as a Practical Baseline

Start with simple descriptors and pretrained embeddings before committing to heavier fine-tuning.

In the COMP30027 image classification project, the strongest workflow combined HOG features, low-level image statistics, and EfficientNet-V2-L embeddings with classical models.

This baseline is useful because it is fast to iterate, easy to evaluate with stratified validation, and strong enough to reveal whether the dataset needs model complexity or better error analysis.
